# Market Research Data Automation Tool

This Python-based utility automates the cleaning, categorization, and pivoting of raw market research data. Developed during my time at Dentsu, the tool was created to replace a manual data preparation process that previously required approximately 30 hours of work each month. 

The tool processes raw Excel sheets and outputs structured, PowerBI-ready CSV files.

## Features

* **Automated Sheet Parsing:** Automatically identifies data frames from raw Excel inputs.
* **Variable Categorization:** Detects and distinguishes between quantitative and categorical variables.
* **Dynamic Pivoting:** Reshapes data into flat, tabular formats suitable for PowerBI and PowerPoint visualization.
* **Custom Selection:** Allows users to select specific data frames and variables to include in the final export.
* **CSV Export:** Generates clean, standardized CSV files ready for downstream BI tools.
